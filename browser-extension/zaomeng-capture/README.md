# 造梦AI 全网采图助手

## 当前能力

- 支持全网图片右键保存到造梦AI素材库
- 支持大图 hover 辅助按钮采集
- 不再局限淘宝 / 天猫，只要浏览器能识别到图片地址就可以采集
- 插件图标使用 `icons/zaomeng-logo.jpg` 生成的多尺寸图标

## 安装方式

1. 打开 `chrome://extensions/`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录 `browser-extension/zaomeng-capture`

## 使用方式

1. 先登录造梦AI网站并打开素材库页面
2. 打开任意网页
3. 在想采集的图片上右键，或先把鼠标移到目标图片上再右键
4. 点击“保存至造梦AI”
5. 回到造梦AI页面查看采集结果

说明：部分网站的大图不是标准 `<img>`，而是背景图或带遮罩的图片。插件菜单现在会在页面右键里始终显示，并优先使用右键命中的图片；如果浏览器没有给出图片地址，会尝试保存鼠标最后悬停到的大图。Chrome 右键菜单不支持在文字中嵌入自定义图片，菜单左侧显示的是扩展图标。

## 备用方式

- 鼠标移到较大的图片上，会出现“采集到造梦AI”悬浮按钮
- 如果悬浮按钮被网页遮挡或不稳定，优先使用右键保存

## 更新插件

修改插件文件后，需要打开 `chrome://extensions/`，点击该扩展的刷新按钮，或重新加载已解压的扩展程序。
