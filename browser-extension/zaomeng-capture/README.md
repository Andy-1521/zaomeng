# 造梦AI 全网采图助手

## 当前能力

- 支持全网图片右键保存到造梦AI素材库
- 支持大图 hover 辅助按钮采集
- 支持常见 `<img>`、`srcset`、懒加载 data 属性和背景图结构识别
- 点击浏览器工具栏插件图标，可打开或聚焦造梦AI工作台
- 不再局限淘宝 / 天猫，只要浏览器能识别到图片地址就可以采集
- 插件图标使用 `icons/zaomeng-logo.jpg` 生成的多尺寸图标

## 支持浏览器

- Chrome
- Edge
- Brave
- Arc
- 360 极速浏览器

说明：以上浏览器均基于 Chromium 内核，可加载同一套 Manifest V3 插件包。Firefox 和 Safari 暂未提供安装包。

## 安装方式

1. 下载 zip 后解压，得到 `zaomeng-capture` 目录
2. 打开扩展管理页：Chrome / Brave / Arc 使用 `chrome://extensions/`，Edge 使用 `edge://extensions/`
3. 开启右上角“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择解压后的 `zaomeng-capture` 目录
6. 刷新造梦AI页面，确认导航栏显示“插件已连接”

360 极速浏览器通常也使用 Chromium 扩展管理页；如果地址栏不接受 `chrome://extensions/`，请从浏览器菜单进入“扩展程序 / 扩展管理”。

## 使用方式

1. 先登录造梦AI网站并打开工作台或素材库页面
2. 打开任意网页
3. 在想采集的图片上右键，或先把鼠标移到目标图片上再右键
4. 点击“保存至造梦AI”
5. 也可以点击图片右上角悬浮的“采集”按钮
6. 回到造梦AI页面查看采集结果

说明：部分网站的大图不是标准 `<img>`，而是背景图、srcset、懒加载 data 属性或带遮罩的图片。插件菜单会在页面右键里始终显示，并优先使用右键命中的图片；如果浏览器没有给出图片地址，会尝试保存鼠标最后悬停到的大图区域。Chrome 右键菜单不支持在文字中嵌入自定义图片，菜单左侧显示的是扩展图标。

## 常见问题

### 导航栏一直显示“插件未连接”

1. 确认扩展已启用
2. 开启右上角“开发者模式”
3. 刷新造梦AI页面
4. 如果网站域名变过，重新从当前站点下载插件包并重新加载

### 保存时报未登录

1. 先登录造梦AI网站并打开素材库页面
2. 确认下载插件包的网站域名和当前登录的网站域名一致
3. 重新刷新造梦AI页面后再采集

### 右键没有识别到正确图片

1. 把鼠标停在大图区域再右键
2. 使用悬浮“采集”按钮
3. 如果图片来自登录后或防盗链资源，保持原网页不要关闭后再采集

## 备用方式

- 鼠标移到较大的图片上，会出现“采集到造梦AI”悬浮按钮
- 如果悬浮按钮被网页遮挡或不稳定，优先使用右键保存

## 更新插件

重新下载 zip 并解压覆盖旧目录，然后打开扩展管理页，点击该扩展的刷新按钮，或删除后重新“加载已解压的扩展程序”。
