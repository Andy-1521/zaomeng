# 造梦AI 商品采图助手

## 当前能力

- 支持淘宝 / 天猫页面图片 hover 采集
- 点击图片右上角“采集到造梦AI”按钮后，将图片发送到已登录的造梦AI网站账户

## 安装方式

1. 打开 `chrome://extensions/`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择当前目录 `browser-extension/zaomeng-capture`

## 使用方式

1. 先登录造梦AI网站并打开“模板提取”页面
2. 打开淘宝 / 天猫商品页面
3. 鼠标移到商品图上方
4. 点击“采集到造梦AI”按钮
5. 回到造梦AI页面查看采集结果
